UBC Card classes

  • .ubc-card - outside wrapper identifier
    • .ubc-card__primary-action - the primary content wrapper that usually contains a single action - ie. clicking through to a page or form. This is often on an a tag wrapping the entire card content.
      • .ubc-card__media - a wrapper for optional media in the card (image / video)
        • .ubc-card__media-content - an optional content wrapper within the media region
      • .ubc-card__primary-content - a wrapper for the primary content (card title, text) this can be contained within the .ubc-card__media if needed for display purposes
      • .ubc-card__secondary-content - a wrapper for an optional region to break apart content into separate regions. For example, .ubc-card__primary-content wraps the card title inside .ubc-card__media so the title can be displayed over top of an image, and .ubc-card__secondary-content contains card text outside the .ubc-card__media region
    • .ubc-card__actions - an optional region outside of the primary action for adding additional actions - ie. share, other related links
